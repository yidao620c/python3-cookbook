============================
5.2 打印输出至文件中
============================

----------
问题
----------
你想将print()函数的输出重定向到一个文件中去。

|

----------
解决方案
----------
在print()函数中指定 ``file`` 关键字参数，像下面这样：

.. code-block:: python

    with open('somefile.txt', 'rt') as f:
        print('Hello World!', file=f)

----------
讨论
----------
关于输出重定向到文件中就这些了。但是有一点要注意的就是文件必须是以文本模式打开。
如果文件时二进制模式的话，打印就会出错。
